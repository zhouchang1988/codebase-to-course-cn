**选择适合你课程版本的章节：**
- 普通版：关注"教学弧线"、"代码↔英语翻译"、"交互元素"
- 专业版：关注"教学目标"、"设计意图说明"、"架构图/数据流图"

### 交互元素

检查此模块需要哪些元素。包含足够的细节供写作代理构建它们。

**普通版交互元素：**
- [ ] **代码↔英语翻译** — 使用上面哪些片段
- [ ] **测验** — [数量] 个问题，风格：[场景 / 调试 / 架构 / 追踪]。每个问题角度的简要描述。
- [ ] **群聊动画** — 角色：[列表]。消息流摘要：[谁对谁说什么，按什么顺序]
- [ ] **数据流动画** — 角色：[列表]。步骤：[高亮和数据包移动的序列]
- [ ] **拖放** — 项目：[列表]，目标：[列表]
- [ ] **其他** — [架构图、层次切换、模式卡片等]

**专业版交互元素：**
*注意：测验默认不包含，仅在用户明确要求时添加。*
- [ ] **代码分析块** — 使用上面哪些片段，重点解释什么设计决策
- [ ] **架构图** — [类型：组件图 / 层次图 / 部署图]。主要组件：[列表]。关键标注：[信息]
- [ ] **数据流图** — [角色和步骤，数据包移动序列]
- [ ] **时序图/状态图** — [按需：展示交互或状态转换]
- [ ] **深度测验** — （可选，仅用户要求时）[数量] 个问题，类型：[设计决策 / 场景应用 / 问题诊断]
